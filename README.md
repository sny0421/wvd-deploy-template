# wvd-deploy-template


## 必須構成
- 仮想ネットワーク
- Azure ADDS or WSAD + AADC
- FileServer

## オプション構成
- Azure Bastion
- Azure Firewall
- Azure NetApp Files
- Virtual Network Gateway

4. カスタムイメージ展開
5. 仮想ネットワーク展開
6. Azure ADDS 展開
7. ドメイン参加用ユーザー作成
